## Definitions

* **Mold** - Refers to the specific shape of a brick, no matter its color. So a _2x2 brick_ will always be considered the same mold, no matter if it's a yellow, red, or any other colored brick.
* **Brick** - A brick is the combination of mold and color. A _2x2 red brick_ is considered a different item than a _2x2 blue brick_.
* **Piece** - This means a specific, physical item. If a set contains 2 _2x2 red bricks_, these are considered two different pieces.
